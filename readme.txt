Easily switch currencies in WooCommerce with custom manual rates, session-based selections, and gateway restrictions.

== Description ==
ThemexHunt Currency Switcher for WooCommerce allows your store to support multiple currencies with manual rate controls. Users can switch currencies seamlessly, and store owners can define gateway restrictions based on currency preferences. This lightweight plugin integrates directly into WooCommerce and includes a shortcode-based dropdown for currency switching.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to **WooCommerce > Currency Switcher** to configure settings

== Frequently Asked Questions ==

= Is this plugin compatible with the latest version of WooCommerce? =
Yes, it is tested up to WooCommerce 8.5 and WordPress 6.8.

= Does this plugin automatically fetch exchange rates? =
No. You can manually define currency rates in the admin panel.

= Can I restrict payment gateways per currency? =
Yes. You can configure allowed gateways per currency under settings.
